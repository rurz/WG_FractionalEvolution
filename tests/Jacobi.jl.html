Testing Julia packages and methods for producing Jacobi functions used to represent the eigenfunctions of the discrete and finite harmonic oscillator

Theory

In their 2016 article, Steffen Weimann et. al [1], used the eigenfunctions of the angular momentum coupling J^x to represent the modes of the finite and discrete harmonic oscillator as

unq=2n(j−n)!(j+n)!(j−q)!(j+q)!Pnq−n,−q−n(0),

where Pnα,β(0) is the Jacobi function at z=0; n and q are the energy levels and position index, respectively such that n,q∈{−j,...,j}, for j∈Z+.

The Task

For the purpose, we want to build a dj×dj matrix for dj=N+1=2j+1 where the entries are labeled by the pair (n,q). So, we need the numerical evaluation of the Jacobi function around z=0. __________

(I) Using HypergeometricFunctions.jl

Since this package hasn't implemented the function Pnα,β(z), we use the definition in terms of the hypergeometric function 2F1(n,a,b;c) as

Pnα,β(z)≡(n+αn)2F1(−n,−n+α+β+1,α+1;12(1−z)).

Where we expect a nicely behaviour around z=0. The issue as you can see is that α,β>−1, who is something not fullfilled when α=−q−n.

(III) Using series representation expansion around z=0

Provided that we can obtain a closed form of Pnα,β(z) for z→0, we use this approach. Given in Wolfram Function Library, the definition used is

Pnα,β(z)=2−n∑k=0n(α+nk)(β+nn−k)(z+1)k(z−1)n−k,

who isn't ill-defined at z→0, and we expect to behave nicely for values α,β<−1.
